# CS_3090_Block_2_Project
This repository contains a password strength checker done for educational purposes. DO NOT USE THIS IN ANY ACTUAL SECURITY CAPACITY.

Program code is in two cs files: PasswordStrengthChecker.cs and PasswordInputer.cs.

This program, accessed by running the PasswordInputer.cs file, will allow the user to input a password and immediately get feedback on its strength.
A password's strength is either "Sufficient" or "Insufficient" and depends on whether it follows these rules:
1. A password must be one continuous string.
2. A password's length must be at least 10 characters long.
3. A password SHOULD be made up of at least a quarter numbers along with another quarter being non-letter or non-number characters.

Again, DO NOT use this tool for securing sensitive information. It was made for educational purposes only.
